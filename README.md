# SMART PANEL
Smart Panel is an advanced video player that allows retrieving files from a cloud directory and playing music via Spotify. With a friendly and intuitive interface, Smart Panel offers a comprehensive solution for media management and viewing.

### Dependencies
Python 3.7 or higher
Libraries: tkinter, selenium, vlc, requests, pyautogui, among others.
